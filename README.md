# Push_Swap
This project has been created as part of the 42 curriculum by rdinis

## Description
Push Swap is a project that sort a list of number>
This project use the Radix Sort
## Instructions
1. without the checker
   - make
     - ./push_swap 1 5 4 3 6
2. with the checker
   - make
     - ARG="4 67 3 87 23"; ./push_swap $ARG | ./checker_OS $ARG
